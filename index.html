Creative Stash (2020)

Creative Stash is a project about sharing creative ideas with the community, users can register, personalise their profile, add ideas, comment and like.

StockTweets (2021)

StockTweets is a service that combines data between Finnhub, which offers real-time stocks, and Twitter, a social network service where a user can write messages.

The service provides information about the current stock market and what discussions are taking place around a particular stock, hence the connection to Twitter.

SpaceX (2022)

SpaceX is the interpretation of the private space travel company of Elon Musk. The structure of the platform is based on JavaScript’s library React. The web application is based on the external SpaceX-API.

SpaceX provides information about the launches of various rockets, both upcoming and completed ones. In turn, the user can get additional background information about each launch and rocket.

byCloudberry (2023)

byCloudberry is a Swedish handbag and accessory company focusing on sustainability in terms of using leftover materials in their products. The founder of byCloudberry wished to implement a functionality in each product page to showcase the level of sustainability for each specific product.

The primary source is WordPress and also additional resources were used as WordPress could not provide all the functionalities that were needed. Render acts as the unified cloud hosting service (that this project is built and run on). MongoDB acts as the plugin that is created for byCloudberry’s website. In other words, a database that is hosted at MongoDB.com.

PickPanter (2023)

Pickpanter is a project about increasing the recycling of PET bottles and cans in Sweden. As the mockups show (created in both Figma and Adobe XD) the meaning of PickPanter is to unite private individuals who want to donate and receive PET bottles and cans.
